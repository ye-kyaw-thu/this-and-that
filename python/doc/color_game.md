# Running example

Round သုံးခုပြီးတာနဲ့ ပရိုဂရမ်ကို ရပ်စေချင်ရင် -n 3 ဆိုတဲ့ parameter နဲ့ run ပါ။  
default က -n 10 နဲ့ color name option နဲ့ run ပေးလိမ့်မယ်။  

```
python ./color_game.py -n 3
```
